# AWS multi-tenat saas platform using terraform

## Project overview 
This project demonstrates how to build a mutli-tenant saas platform on AWS using terraform 

## What is multi-tenant ?
Multi-tenant architecture allows multiple customers(tenants) to use the same application while keeping their data isolated.


Example:
- tenant1.app.com
- tenant2.app.com

## Architecute compontents 
- AWS VPC (Networking)
- ECS Farget (Application Hosting)
- Application Load balancer (Traffic Routing)
- RDS PostgreSQL (Database)
- Route53 (DNS)
- Secret manager (Secure credentials)

## Project Goals 
- Learn Terraform Modules 
- Build real-world Saas Infrastructure 
- Understand AWS Networking and service 
- Implement tenent isolation
